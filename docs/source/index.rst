=================================
Welcome to PyZPK!
=================================

PyZPK is a Python wrapper for open-source Zero Proof Knowledge Library, libsnark.
This library provides a set of zkSNARK schemes, a cryptographic method for
proving/verifying, in zero-knowledge, the integrity of computations.
